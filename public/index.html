dead-drop.io

is a place for media dead-drops. To use it, simply drag and drop a supported file onto this page, or copy and paste a media url into the box. When you share a piece of media, you will get to see the previous users share. No files are stored, its a one time anonymous share for the next user. You give one, you get one.

dead-drop.io a collaborative project of:
Derek Anderson & Ben Moren

Suported File Formats:
.gif .jpg .png .bmp .txt .csv .md
Linked file Formats:
Vimeo, Youtube, SoundCloud, BandCamp
Filesize Limit: 25mb
